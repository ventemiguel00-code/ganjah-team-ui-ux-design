2. Abre PowerShell en la carpeta del proyecto


1. Descarga todos los archivos del proyecto
2. Abre Terminal en la carpeta del proyecto
3. Ejecuta:
   ```bash
   chmod +x SUBIR_A_GITHUB.sh
   ./SUBIR_A_GITHUB.sh
   ```
